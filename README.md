# The Letter K
A text-based Python project where the user's goal is to find the target item
containing the letter K within a list of distractors. There are 8 trials,
each presented only once, with difficulty increasing each round. Difficulty
varies by item length, list length, or distractor features.
# Inspiration
This project is inspired by the work of Ulric Neisser, a notable cognitive
psychologist, particularly his 1964 paper on visual search. In it, Neisser
studied how people perceive and process visual information amid distractors or irrelevant information
by having participants scan long lists until they found the target items.
# Output
At the end, the user receives a summary of their results, including
a graph of reaction time for each trial.
